# Placeholder pour résultats de benchmarks

Les fichiers générés par `scripts/benchmark.sh` seront sauvegardés ici.

Format: `benchmark-YYYYMMDD-HHMMSS.md`

## Exemple

```bash
cd ~/equipment-ponyo
bash scripts/benchmark.sh
```

Le rapport de benchmark (CPU, RAM, disque, GPU) sera généré dans ce dossier.
